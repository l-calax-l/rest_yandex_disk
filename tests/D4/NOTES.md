# D4 — Тестирование API (CRUD папок)


## Что сделано

- TC-D4-01: Создание новой папки (PUT /resources)
- TC-D4-02: Перманентное удаление папки (DELETE permanently=true)
- TC-D4-03: Восстановление папки из корзины (PUT /trash/resources/restore)
- TC-D4-04: Создание папки с существующим именем → 409
- TC-D4-05: Удаление несуществующей папки → 404
- TC-D4-06: Восстановление несуществующего ресурса → 404

## Что обнаружил

- Корзина добавляет суффикс к имени:
   `TestFolder_abc123` → `trash:/TestFolder_abc123_f15f6ffbc0466b...`
- Удаление и восстановление — асинхронные:
   Нужно ожидание (wait_for_result) с таймаутом
- `wait_for_result()` — утилита для ожидания асинхронных операций
